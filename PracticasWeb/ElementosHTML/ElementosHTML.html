En este tutorial se describirán algunos de los elementos y atributos HTML, es decir, se mencionarán características de los elementos y la manera de usarse a modo de tutorial.

Elemento H1 - Encabezados

Los elementos de encabezado implementan seis niveles de encabezado del documento, <h1> es el más importante, y <h6>, el menos importante. Un elemento de encabezado describe brevemente el tema de la sección que presenta. La información de encabezado puede ser usada por los agentes usuarios, por ejemplo, para construir una tabla de contenidos para un documento automáticamente.

El siguiente código muestra cómo usar los encabezados

<h1>Texto 1</h1>

<h2>Texto 2</h2>

<h3>Texto 3</h3>

<h4>Texto 4</h4>

<h5>Texto 5</h5>

<h6>Texto 6</h6>

Este es el resultado

Texto 1

Texto 2

Texto 3

Texto 4

Texto 5
Texto 6

Elemento P - Párrafos

El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos. Sus características son:

  • Sus etiquetas son: <p> y </p> (la de cierre es opcional).
  • Crea una caja: en bloque.
  • Está definido como: Elemento de frase.
  • Puede contener: cero o más elementos en linea.

El siguiente código muestra cómo usar los párrafos

<p>Este es el primer párrafo.</p>

<p>Este es el segundo párrafo.</p>

Este es el resultado

Este es el primer párrafo.

Este es el segundo párrafo.

Elemento IMG - Imágenes

El elemento de imagen HTML <img> representa una imagen en el documento.

El siguiente código muestra cómo usar las imágenes

<img src="imagenes/imagen.jpg" style="display: block; margin-left: auto; margin-right: auto;">

Elemento A - Ancla

El Elemento HTML Anchor <a> crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.

El siguiente código muestra cómo usar las anclas

<p>Se puede redirigir de diferentes maneras, se puede dirigir a un sitio web como
<a href="https://www.google.com">Google</a>,
alguna otra sección del mismo sitio como a los <a href="#parrafos">párrafos</a>,
e incluso a un <a href="mailto:carlos.medina18@tectijuana.edu.mx">correo electrónico</a>.</p>

Este es el resultado

< style="color: red;">Atributo SRC - Origen

Este atributo es obligatorio para las imágenes (elemento img), por lo que su uso no tiene mucha ciencia, mas que el origen puede ser relativo o absoluto dependiendo de la fuente de la imagen.

El siguiente código muestra cómo usar el atributo

<img src="imagenes/imagen.jpg" alt="Imagen que toma directamente de este sitio">

Atributo STYLE - Estilo

Este atributo existe como alternativa del elemento style de HTML y el archivo de CSS (referido desde un HTML con el elemento link), en caso específico en el que se quiera dar un estilo único a un elemento en concreto. Se escribe exactamente lo mismo que se escribiría dentro de una etiqueta style o de un archivo CSS, pero con la diferencia de que se escribe todo en una sola línea.

El siguiente código muestra cómo usar el atributo

<h1 style="color: red; text-align: right;">Titulo 1</h1>

Este es el resultado

Titulo 1

Atributo HREF - Referencia de hipervínculo

Este atributo es obligatorio para el elemento de ancla (a), ya que este atributo es el que le da sentido al elemento, en este atributo es donde se escribe la referencia relativa o absoluta a donde redirigirá el elemento; ergo, se puede escribir tanto un directorio dentro del sitio, una sección de la misma página u otra, o un url de un sitio externo.

El siguiente código muestra cómo usar el atributo

<p>Este texto te puede redirigir tanto al sitio de <a href="https://www.google.com">Google</a> como a la sección de <a href="#parrafos">párrafos</a> que se encuentra arriba de esta misma página.</p>

Este es el resultado

Este texto te puede redirigir tanto al sitio de Google como a la sección de párrafos que se encuentra arriba de esta misma página.
