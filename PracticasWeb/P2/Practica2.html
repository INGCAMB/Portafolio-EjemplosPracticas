    1. Red computacional:Conjunto de computadoras conectadas entre si para el intercambio y administracion de datos a travez de esta.

      2. Regresar






    2. Internet:Red de equipos de computo que se comunican etre si empleando un lenguaje comun.

      3. Regresar






    3. Navegador:Programa que permite vizualizar paginas de internet.

      4. Regresar






    4. Buscador:Pagina de internet que te permite buscar paginas de internet.

      5. Regresar






    5. HTTP y HTTPS:HTTP Hyper Text Transfer Protocol Protocolo de red para publicar paginas de internet, HTTPS indica que se esta usando proteccion al transferir informacion.

      6. Regresar






    6. HTML:Hyper Text Markup Language lenguaje usado por los navegadores para mostrar paginas de internet.

      7. Regresar






    7. URI y URL:URI Uniform Resource Identifire Identifica recursos de internet URL Uniform Resource Locator Nombra recursos en internet.

      8. Regresar






    8. Dominio de internet:Nombre que sirve para identificar direcciones de computadoras conectadas a internet.

      9. Regresar






    9. Direccion IP:Numero dividido en 4 partes usado como identificador unico.

      10. Regresar






    10. Router:Hardware que funciona como semaforo para controlar las señales que trafican dentro de una red.

      11. Regresar






    11. VPN:Virtual Private Network Red de comunicaciones inalambricas que usa infraestructuras de internet para proveer conexiones remotas a redes de una organizacion.

      12. Regresar






    12. Ancho de banda:Medida que define la velocidad de tu conexion a internet.

      13. Regresar






    13. Escalabilidad:Propiedad de aumentar la capacidad de tamaño de un sistema sin comprometer su funcionamiento y calidad normales.

      14. Regresar






    14. WiFi:Marca registrada que nombra la tecnologia con la que se conectan dispositivos elctronicos de manera inalambrica.

      15. Regresar






    15. Red social:Pagina web donde se intercambia informacion personal y contenido multimedia.

      16. Regresar






    16. La nube:Almacenamiento masivo de datos en servidores.

      17. Regresar
