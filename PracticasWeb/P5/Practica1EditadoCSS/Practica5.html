Pagina para mas informacion

  1. Red computacional:Conjunto de computadoras conectadas entre si para el intercambio y administracion de datos a travez de esta.

  2. Internet:Red de equipos de computo que se comunican etre si empleando un lenguaje comun.

  3. Navegador:Programa que permite vizualizar paginas de internet.

  4. Buscador:Pagina de internet que te permite buscar paginas de internet.

  5. HTTP y HTTPS:HTTP Hyper Text Transfer Protocol Protocolo de red para publicar paginas de internet, HTTPS indica que se esta usando proteccion al transferir informacion.

  6. HTML:Hyper Text Markup Language lenguaje usado por los navegadores para mostrar paginas de internet.

  7. URI y URL:URI Uniform Resource Identifire Identifica recursos de internet URL Uniform Resource Locator Nombra recursos en internet.

  8. Dominio de internet:Nombre que sirve para identificar direcciones de computadoras conectadas a internet.

  9. Direccion IP:Numero dividido en 4 partes usado como identificador unico.

  10. Router:Hardware que funciona como semaforo para controlar las se単ales que trafican dentro de una red.

  11. VPN:Virtual Private Network Red de comunicaciones inalambricas que usa infraestructuras de internet para proveer conexiones remotas a redes de una organizacion.

  12. Ancho de banda:Medida que define la velocidad de tu conexion a internet.

  13. Escalabilidad:Propiedad de aumentar la capacidad de tama単o de un sistema sin comprometer su funcionamiento y calidad normales.

  14. WiFi:Marca registrada que nombra la tecnologia con la que se conectan dispositivos elctronicos de manera inalambrica.

  15. Red social:Pagina web donde se intercambia informacion personal y contenido multimedia.

  16. La nube:Almacenamiento masivo de datos en servidores.

  17. SSL:Secure Sockets Layer Protocolo que permite que las aplicaciones transmitan informacion de forma segura.

  18. Blog:Pagina web con estructura cronologica actualizable que trata un tema en especifico.

  19. ISP:Internet Sevice Provider Proveedor de servicios de internet por una cuota mensual.

  20. P2P:Peer-to-peer Red sin clientes ni servidores fijos.

  21. Streaming:Escuchar musica o ver videos sin descargarlos antes de verlos.

  22. IM:Mensajeria instantanea, comunicacion en tiempo real basada en texto.

  23. VoIP:Voice over internet protocol hardware y software que permite usar internet como transmision de llamadas telefonicas.

  24. Malware:Malicious software Programa o codigo malicioso que da単a un sistema o logra un mal funcionamiento.

  25. Phishing:Captacion de datos personales de manera ilicita por internet.

  26. Troyano:Software malicioso que se presenta como un programa inofensivo pero que al ejecutarlo brinda acceso remoto al equipo.

  27. Spyware:Software que recopila informacion de un ordenador y despues la transmite sin concentimiento del propietario.

  28. Download:Copiar datos de una fuente principal a un dispositivo periferico.

  29. DRM:Digital Right Managament Medio que controla la distribucion de archivos digitales.

  30. SPAM:Correo no deseado que se envia a muchas personas con fines publicitarios.

  31. FTP:File Transfer Protocol Transfiere grandes bloques de datos por la red.
