Tutorial de elementos basicos: Tabla HTML

Medina Beltran Carlos Alberto

En este tutorial se describirán algunos de los elementos y atributos de tablas HTML, es decir, se mencionarán características de los elementos y la manera de usarse a modo de tutorial.

Denifir una tabla HTML

Una tabla en HTML consta de celdas de tabla dentro de filas y columnas.

El siguiente código muestra cómo crear una tabla

<table style="width:100%">

<tr>

<th>Encabezado 1</th>

<th>Encabezado 2</th>

<th>Encabezado 3</th>

</tr>

<tr>

<td>Texto</td>

<td>Texto</td>

<td>Texto</td>

</tr>

<tr>

<td>Texto</td>

<td>Texto</td>

<td>Texto</td>

</tr>

</table>

Este es el resultado

Encabezado 1 Encabezado 2 Encabezado 3
Texto Texto Texto
Texto Texto Texto

Celdas de tablas

Cada celda de la tabla esta definida por una etiqueta <td> y </td>.

  • Todo entre <td> y </td> es el contenido de la celda de la tabla.
  • td significa datos de tabla.
  • Nota: Los elementos de datos de la tabla son los contenedores de datos de la tabla.
  • Puede contener todo tipo de elementos HTML: texto, imágenes, listas, otras tablas, etc.

El siguiente código muestra cómo usar las celdas

<table style="width:100%">

<tr>

<td>Alto</td>

<td>Chico</td>

<td>Mediano</td>

</tr>

</table>

Este es el resultado

Alto Chico Mediano

Filas de tablas

Cada fila de la tabla comienza con una etiqueta <tr> y termina con una etiqueta </tr>.

  • tr significa fila de tabla.
  • Nota: Hay ocasiones en las que una fila puede tener menos o más celdas que otra.
  • Puede tener tantas filas como desee en una tabla, solo asegúrese de que la cantidad de celdas sea la misma en cada fila.

El siguiente código muestra cómo usar las filas

<table style="width:100%">

<tr>

<td>Alto</td>

<td>Chico</td>

<td>Mediano</td>

</tr>

<tr>

<td>22</td>

<td>21</td>

<td>20</td>

</tr>

</table>

Este es el resultado

Alto Chico Mediano
22 21 20

Encabezados de tablas

A veces desea que sus celdas sean encabezados, en esos casos use la etiqueta <th> en lugar de la etiqueta <td>.

  • De manera predeterminada, el texto en los elementos <th> está en negrita y centrado, pero puede cambiar eso con CSS.

El siguiente código muestra cómo usar los encabezados de las tablas

<table style="width:100%">

<tr>

<th>Rayo</th>

<th>Alberto</th>

<th>Carlos</th>

</tr>

<tr>

<td>Alto</td>

<td>Chico</td>

<td>Mediano</td>

</tr>

<tr>

<td>22</td>

<td>21</td>

<td>20</td>

</tr>

</table>

Este es el resultado

Rayo Alberto Carlos
Alto Chico Mediano
22 21 20
