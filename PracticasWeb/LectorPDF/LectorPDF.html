Cómo hacer un lector de PDF en la página HTML

Para hacer un lector de PDF dentro de nuestro sitio actualmente es sencillo puesto que existen servicios de terceros que nos proporcinan lo necesario para poder llevar a cabo lo que buscamos.

En este caso vamos a utilizar el lector PDF que nos proporciona Google, que para acceder a este, se tiene que manejar la etiqueta <iframe>.

Tenemos que tomar en cuenta que en la etiqueta se van a utilizar los siguientes atributos:

  • scr: Aquí se pondrá la liga de el lector de Google, y seguido se pondrán 2 propiedades extras DENTRO de la misma liga; estos son la url que tendrá la liga del pdf que queremos visualizar, y una última propiedad llamada embedded que este forzosamente tendrá el valor de "true" para que funcione el visualizador
  • style: Se necesita poner estilo CSS al iframe para determinar el tamaño específico del contenedor; es decir que tendrá las propiedades de width y height.
  • frameborder: Este atributo es opcional, pero se agrega en caso de que se le quiera dar un poco más de estilo al contenedor.

Para explicar cómo extraer el lector a nuestro sitio, necesitamos seguir la siguiente sintaxis de la etiqueta.

<iframe src="http://docs.google.com/gview?url={LigaDeTuPdf}&embedded=true" style="width:100%; height:700px;" frameborder="1"></iframe>

Si el archivo es local pones en SRC el directorio si no tienes que poner url activo.

Nótese que en src se escribe primero la dirección del visor de google, y se le manda como parametros nuestra liga de nuestro pdf y después el embedded=true, seguido tenemos los estilos, los cuales se recomiendan esas propiedades puestas, pero pueden ser modificables, admeás el frameborder que ya se mencionó que es completamente opcional.
